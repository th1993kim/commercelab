# AI 작업 지침

이 프로젝트에서 AI는 작업을 시작하기 전에 다음 문서를 우선 확인합니다.

- `README.md`
- `docs/conventions/backend-code-convention.md`

기본 원칙은 다음과 같습니다.

- 백엔드 코드는 Kotlin으로 작성합니다.
- 백엔드 구조는 헥사고날 아키텍처를 기준으로 설계합니다.
- 기존 문서와 코드 컨벤션을 우선합니다.
- 도메인 경계를 임의로 넘지 않습니다.
- 테스트나 실행 결과 없이 동작을 단정하지 않습니다.
- 설계 판단이 필요한 변경은 README, ADR, 관련 문서에 기록합니다.
